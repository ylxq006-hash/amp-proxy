# Amp-Prox 代理工具

Amp CLI 的专业包装器，支持自定义后端并修复了官方客户端的启动崩溃问题。

## 核心特性
- **配置分离**: 所有设置均在 `config/config.json` 中，无需改动代码。
- **自动启动代理**: 运行命令时自动检查并启动后台代理服务。
- **时间戳日志**: 详细运行日志保存在 `logs/` 目录，并自动滚动清理旧日志。
- **崩溃修复**: 内置 Mock 数据逻辑，确保 CLI 在各种环境下都能稳定运行。

## 目录结构
- `bin/`: 存放可执行脚本。
- `config/`: 存放用户配置文件。
- `src/`: 存放代理服务器核心代码。
- `logs/`: 存放代理运行日志。

## 安装与初始化
在项目目录下运行：
```bash
npm run setup
```
这会自动建立全局软链接。

## 使用方法
直接在终端输入命令：
```bash
amp-prox
```
